# FPGA Digital Design & VHDL Implementations

This repository contains a collection of fundamental digital hardware designs and Register-Transfer Level (RTL) implementations using VHDL. These modules serve as the essential building blocks for more complex FPGA and ASIC architectures.

## 📂 Hardware Modules Included
* **Sequential & Combinational Logic:** Registers, Shift Registers, Binary Counters, and Edge Detectors.
* **Arithmetic & Data Path:** Universal Adders, Sign Adders, and BCD (Binary-Coded Decimal) processing.
* **Memory & Timing:** FIFO Buffers, Square Wave Generators, and Multiplexed Display Controllers (LED Time Mux).
* **System Integration:** A complete digital Stopwatch implementation.

## 🛠️ Technologies & Focus
* **Language:** VHDL
* **Concepts:** Digital Logic Design, Hardware Description, RTL Coding, Synchronous/Asynchronous Circuits.
